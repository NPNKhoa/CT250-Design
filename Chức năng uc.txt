- Khách vãng lai:
+ Đăng ký tài khoản
+ Tìm kiếm sản phẩm
	+ Tìm kiếm theo giọng nói
	+ Tìm kiếm theo từ khóa
+ Xem thông tin sản phẩm
	+ Lọc theo thương hiệu
	+ Lọc theo giá
+ Xem bài viết

- Khách hàng là thành viên hệ thống:
+ Đăng nhập
+ Đăng xuất
+ Quản lý giỏ hàng
	+ Thêm sản phẩm vào giỏ hàng
	+ Cập nhật số lượng sản phẩm
	+ Xóa sản phẩm khỏi giỏ hàng
+ Lấy mã giảm giá
+ Xem thông tin cá nhân
	+ Cập nhật thông tin cá nhân
+ Thanh toán
+ Đặt hàng
+ Quản lý đơn hàng
	+ Xem chi tiết đơn hàng
		+ Hủy đơn hàng
	+ Xem lịch sử mua hàng
+ Phản hồi thông tin
	+ Phản hồi sản phẩm
	+ Phản hồi dịch vụ
+ Xem gợi ý sản phẩm
